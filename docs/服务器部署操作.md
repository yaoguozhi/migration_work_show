#### 1.将要部署的压缩包copy到服务器上

```
// 1.复制当前桌面上的public.zip文件到指定用户名的ip地址所要放的文件夹下
scp /Users/tyler.xing/Desktop/public.zip auser@10.11.12.71:/var/www/telecom

// 2.此时提示输入密码
输入密码：Cloudwise$123456

// 3.以上步骤完成本地压缩包到服务器地址的拷贝
```
#### 2.进入服务器地址

```
// 1.连接vpn (vpn连接软件：Tunnelblick)
用户名：howard.gu  密码： Yunzhihui-1  // 没有vpn账户的找领导申请

// 2.输入以下命令进入服务器
ssh auser@10.11.12.71  // auser为用户名，后面为地址

// 3.提示输入密码
密码: Cloudwise$123456  // 通过拷贝粘贴回车即可

// 备注：如需退出服务器
输入： ctrl + d

```
#### 3.进入指定文件夹去解压缩包

```
// 1.进入服务器地址
输入命令：cd /var/www/项目地址

// 2.解压压缩包
输入命令: unzip 文件名.zip // 文件解压到根目录下，暴露index.html文件

// 3.浏览器地址输入地址进入项目
输入： 10.11.12.71/项目地址

```

#### 4.如果浏览器无法进入指定项目，需要配置服务器如下
```
// 1.进入配置文件地址
输入：cd /etc/h2o

// 2.查看当前文件夹下的配置文件
输入：ls

// 3.编辑配置文件 h2o.conf
输入：vi h2o.conf

// 4.增加paths的配置项
paths:
    "/people":      // 地址栏ip后面跟的地址
    file.dir: /var/www/html     // 对应打开的项目目录

// 5.重启h2o服务器
/etc/init.d/h2o restart  // 整体输入后回车

// 常用命令备注：
编辑命令： 输入：i
编辑后保存命令： 输入： ESC + : + wq + 回车
编辑后不保存命令: 输入： ESC + : + q + 回车
重命名文件夹：输入： mv A B  // 将A重命名为B
删除文件命令： 输入：rm 文件名
删除文件夹命令： 输入： rm -rf 文件夹名称
创建文件夹： 输入： mkdir 文件夹名称
回到跟目录： 输入： cd
注意： cd到根目录后进入某一地址需要输入 ‘‘/’’  cd /var/www, 正常输入 cd www即可

```
